# Prettify JSON

The script prints out the data from a file in a human readable JSON format

# Quickstart

Example of script launch on Linux, Python 3.5:

```bash

$ python pprint_json.py <path to file>
# An output example
{
    "foo": 3,
    "bar": 4
}
```
